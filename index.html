Formación

Oracle Next Education

Actualmente me especializo en desarrollo Front-end en el programa ONE de Oracle.

El programa equipa a los estudiantes con habilidades técnicas y conductuales para actuar como Jr. Developer, un profesional que trabaja para crear y mejorar sitios web, software y aplicaciones, y los prepara para su objetivo profesional.

Front-End

El enfoque de esta formación es la creación de páginas web en donde aprendí la base necesaria para estudiar los frameworks más famosos del mercado.

Profundicé aún más en HTML, CSS y principalmente en JavaScript ya que ese es el lenguaje base de la web. Además pude desarrollar buenas practicas que todo desarrollador Front-End debe de tener para crear proyectos escalables.

Business Agility

En la formación de Business Agility aprendí como tu empresa puede prepararse para un escenario de incertidumbre y cambios constantes, entender cuál es el Mindset Ágil que las empresas necesitan, como estructurar equipos ágiles y como medir resultados tanto para el negocio como para la operación.

Cuando se comienza a ver algunos resultados fruto de las implementaciones Agile, es necesario formar nuevos Agile Coaches para que puedan expandir esa cultura de agilidad, entrenar nuevos equipos y así escalar el ambiente ágil.

React

React es una biblioteca del lenguaje JavaScript y una de las formas más modernas de crear aplicaciones web.

Funciona de forma declarativa, eficiente y flexible, permitiendo la creación de interacciones complejas. Para poder tomar esta formaciónes importante tener una base sólida en HTML, CSS y JavaScript. Conocer Git y versionado de código también será muy valioso.
